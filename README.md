# 化学方程式配平

基于高斯消元法，实现化学方程式配平

可以处理多个方程式叠加的情况

~~~bash
Please input an unbalanced chemical eqution:
Cu + HNO3 -- Cu(NO3)2 + NO + H2O
3Cu + 8HNO3 === 3Cu(NO3)2 + 2NO + 4H2O
~~~

~~~bash
Please input an unbalanced chemical eqution:
Cl2+NaOH=NaCl+NaClO+NaClO3+H2O
3NaClO === 2NaCl + 1NaClO3
1Cl2 + 2NaOH === 1NaCl + 1NaClO + 1H2O
~~~

~~~bash
Please input an unbalanced chemical eqution:
Cu-Au
No solution
~~~
